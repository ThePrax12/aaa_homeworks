# task2

Этот проект содержит функцию для декодирования строки из азбуки Морзе и тесты для проверки её работы.

## Инструкции по запуску тестов

1. Установите `pytest`, если он ещё не установлен:
   ```bash
   pip install pytest
2. Запустите тесты командой 
   ```bash
   python -m pytest -v task2.py
